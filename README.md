# 🏅 App Formulário Esporte

Este projeto consiste em uma aplicação desenvolvida com o objetivo de **coletar informações relacionadas a atividades esportivas por meio de um formulário digital**.

A aplicação foi criada como exercício prático para desenvolver habilidades em **desenvolvimento de aplicações, manipulação de dados e criação de interfaces para entrada de informações**.

---

# 📌 Sobre o Projeto

O **App Formulário Esporte** foi desenvolvido com foco em praticar conceitos de programação aplicados à criação de formulários para cadastro de informações relacionadas a atividades esportivas.

Aplicações baseadas em formulários são amplamente utilizadas para **coletar dados de usuários, realizar cadastros e organizar informações** dentro de sistemas.

O projeto permite registrar dados fornecidos pelo usuário através de campos de entrada e processar essas informações dentro da aplicação.

---

# 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

- JavaScript
- React Native
- Expo

# 📱 Funcionalidades

O aplicativo possui funcionalidades como:

- Cadastro de informações esportivas
- Inserção de dados por meio de formulário
- Interface simples para preenchimento
- Manipulação de dados informados pelo usuário

# ⚙️ Como Executar o Projeto

1. Clone o repositório

```

git clone [https://github.com/JoaoRamponi/appFormularioEsporte](https://github.com/JoaoRamponi/appFormularioEsporte)

```

2. Acesse a pasta do projeto

```

cd appFormularioEsporte

```

3. Instale as dependências

```

npm install

```

4. Execute o projeto

```

npx expo start

```

Depois disso, o aplicativo poderá ser executado utilizando o **Expo Go** ou um **emulador Android/iOS**.
