# 📋 App Formulário de Esporte

Aplicativo desenvolvido em **React Native** com **Expo**, que permite o cadastro de informações relacionadas a esportes através de um formulário interativo.

O objetivo do projeto é praticar conceitos de **desenvolvimento mobile**, manipulação de **inputs**, **componentes visuais** e **armazenamento de dados fornecidos pelo usuário**.

---

# 📱 Funcionalidades

* 📄 Cadastro de informações esportivas
* 🎛️ Seleção de opções através de **Picker**
* 🎚️ Ajuste de valores com **Slider**
* 🔘 Ativação de opções com **Switch**
* 🖼️ Interface simples e intuitiva
* 📲 Aplicação mobile utilizando **React Native + Expo**

---

# 🚀 Tecnologias utilizadas

* **React Native**
* **JavaScript**
* **Expo**
* **React Hooks**
* **Components do React Native**

```bash
npm install @react-native-picker/picker

```

# ▶️ Como executar o projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/JoaoRamponi/appFormularioEsporte.git
```

### 2️⃣ Entrar na pasta

```bash
cd appFormularioEsporte
```

### 3️⃣ Instalar dependências

```bash
npm install
```

### 4️⃣ Iniciar o projeto

```bash
npx expo start
```
