# 🏅 App Formulário Esporte

Aplicação mobile desenvolvida para **coletar informações relacionadas a atividades esportivas por meio de um formulário digital**.

O projeto foi criado como exercício prático para desenvolver habilidades em **desenvolvimento de aplicações mobile, manipulação de dados e criação de interfaces para entrada de informações** utilizando React Native.

---

## 📌 Sobre o Projeto

O **App Formulário Esporte** tem como objetivo praticar a criação de formulários dentro de aplicações mobile, permitindo que usuários insiram dados relacionados a atividades esportivas.

Aplicações baseadas em formulários são amplamente utilizadas em sistemas para **cadastro, coleta de informações e organização de dados**.

Este projeto permite registrar dados informados pelo usuário e processá-los dentro da aplicação.

---

## 🚀 Tecnologias Utilizadas

- **JavaScript**
- **React Native**
- **Expo**

## ⚙️ Como Executar o Projeto

1. Clone o repositório

```bash
git clone https://github.com/JoaoRamponi/appFormularioEsporte
````

2. Acesse a pasta do projeto

```bash
cd appFormularioEsporte
```

3. Instale as dependências

```bash
npm install
```

4. Execute o projeto

```bash
npx expo start
```

Após executar o comando, o aplicativo poderá ser aberto utilizando **Expo Go** no celular ou em um **emulador Android/iOS**.
